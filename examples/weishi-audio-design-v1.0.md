# 莫比乌斯环 · 五阶音景 + 呼吸层 · 独立提案

> **方案**：Web Audio API 纯代码合成 · 零外部依赖 · 零文件体积增加
>
> **原则**：音量控制在-24dB以下，不抢视觉注意力。默认静音，用户主动开启。
>
> **版本**：v1.0 独立提案
>
> **关联**：[五空动画设计书v2.0](#)

---

## 一、音频架构：五个音频层

一个 `AudioContext`，五层独立音量控制：

```
AudioContext
├── 呼吸层（全程 0-50s）         ← 觉知域·4秒呼吸周期
├── 建构层（0-8s）              ← 木·生长·种子破土
├── 流转层（8-20s）             ← 水·流动·粒子穿梭
├── 消散层（20-35s）            ← 金·收敛·三轮体空
└── 渐隐层（35-50s）            ← 金→水·为道日损
```

每层一个 `GainNode`（音量控制）+ 各自的振荡器/噪声源。阶段切换时交叉淡入淡出1.5秒。

## 二、呼吸层（全程）—— 最核心的听觉锚点

### 设计理念

呼吸声不是"背景音乐"，而是**觉知域的画布呼吸在听觉维度的镜像**。视觉层的画布以10秒周期极微明暗变化（±2%），听觉层的呼吸以4秒周期做同样的事情——用户感觉不到它的存在，但关掉它的瞬间会感觉"画布死了"。

### 参数设计

| 参数 | 值 | 说明 |
|:-----|:--|:-----|
| **音源** | 粉色噪声（pink noise）低通800Hz | 粉色噪声比白噪声更柔和，低通过滤高频刺耳 |
| **包络周期** | 4秒 | 吸1.2s → 停0.4s → 呼2.0s → 停0.4s |
| **吸气包络** | 0 → -24dB，1.2s线性上升 | 音量渐强，不突兀 |
| **呼气包络** | -24dB → -36dB，2.0s指数衰减 | 缓慢消逝，不留痕迹 |
| **停顿** | -60dB（几乎无声） | 吸呼之间的自然间隙 |
| **全程音量** | 峰值-24dB | 不抢注意力，关掉它时才有感觉 |
| **相位对齐** | 呼吸周期与画布呼吸同步 | 视觉微明暗 = 听觉微明暗 |

### 呼吸声的心理效应

- 用户不会"听到"呼吸声——就像你不会注意到自己的呼吸，除非有人提醒
- 但关掉呼吸层的瞬间，用户会感觉到"画布变死了"
- 这正是觉知域的本义——容纳一切但从不凸显自己

## 三、五阶音景（四阶段 × 五"空"映射）

### 阶段一：建构（0-8秒）· 木·生长

| 参数 | 值 |
|:-----|:--|
| **音源** | 正弦波432Hz（A4自然调谐）+ 正弦波864Hz（高八度泛音） |
| **包络** | 0 → -20dB（0-5s线性渐强），-20dB保持（5-8s） |
| **泛音比** | 基频80% + 泛音20% |
| **表达** | 种子破土的生长感——低音是根，泛音是芽 |
| **对应空** | —（建构阶段，尚未涉及"空"的操作） |

### 阶段二：流转（8-20秒）· 水·流动

| 参数 | 值 |
|:--|:--|
| **音源** | 双正弦波432Hz + 428Hz（4Hz差拍），叠加 轻量白噪声（-30dB） |
| **差拍频率** | 4Hz —— 产生4次/秒的"波动感"，模拟水流 |
| **噪声** | 白噪声经500Hz低通 → -30dB，模拟远处溪流声 |
| **音量** | -22dB 恒定 |
| **表达** | 粒子在种子层↔现行层之间的双向流动。差拍是水波，噪声是水声 |
| **对应空** | 觉知域·大圆镜智（已全程运行） |

### 阶段三：三轮体空（20-35秒）· 金·消散

| 参数 | 值 |
|:--|:--|
| **音源** | 正弦波432Hz → 864Hz 缓慢滑音（频率线性上升） |
| **滑音时长** | 15秒（20s开始滑，35s到达高八度） |
| **音量包络** | -22dB → -40dB（25s开始衰减，35s几乎无声） |
| **泛音** | 高八度泛音比例从20%→40%（越往上越轻盈） |
| **表达** | 粒子消散——低频向高频滑走，音量渐弱至无。频率上升=升维，音量下降=消散 |
| **对应空** | **三轮体空**（主） + **色空不二**（粒子上旋时泛音短暂闪烁） |

### 阶段四：为道日损（35-50秒）· 金→水

| 参数 | 值 |
|:--|:--|
| **音源** | 仅基频216Hz（432Hz的低八度），偶发泛音余响 |
| **泛音余响** | 每3-5秒随机触发一次短促的648Hz泛音（-35dB，0.3s衰减） |
| **音量** | -30dB 恒定（极弱，几乎听不见） |
| **表达** | 损之又损——频率退回低八度，只剩一根弦在震动。偶发泛音是"还没完全消失的最后一个念头" |
| **对应空** | **为道日损**（主） |

## 五、开关控制

```json
{
  "audio": {
    "enabled": true,
    "breath": true,
    "construct": true,
    "flow": true,
    "dissolve": true,
    "wane": true,
    "baseFreq": 432,
    "masterVolume": 0.06,
    "breathPeriod": 4000
  }
}
```

| 开关 | 关闭效果 |
|:-----|:--------|
| `audio.enabled` | 全部静音 |
| `breath: false` | 无呼吸声（其余四阶保留） |
| `construct: false` | 建构阶段无声 |
| `masterVolume: 0.03` | 整体音量减半 |

## 六、UI 交互

### 右下角音频控件

```
┌─────────────────────┐
│  🔇  🔈  🔊          │  ← 点击切换三态
│  音量滑块 ──●────    │  ← 左右拖动调整
└─────────────────────┘
```

三态：
- 🔇 静音（默认）—— 遵守浏览器自动播放策略
- 🔈 呼吸+音景 —— 完整听觉体验
- 🔊 仅音景（无呼吸）—— 只想听音景时

首次打开页面：显示 🔇。用户点击后切换到 🔈 并启动 AudioContext。

### 浏览器自动播放策略兼容

Chrome/Safari 禁止页面自动播放音频。必须在用户首次交互（点击喇叭图标）时创建 `AudioContext`。这是标准做法，不影响体验。

## 七、技术实现要点

### 7.1 粉色噪声生成

```javascript
// 粉色噪声 = 白噪声经 -3dB/octave 滤波器
function createPinkNoise(ctx) {
  var bufSize = 4096;
  var node = ctx.createScriptProcessor(bufSize, 1, 1);
  var b0=0,b1=0,b2=0,b3=0,b4=0,b5=0,b6=0;
  node.onaudioprocess = function(e) {
    var out = e.outputBuffer.getChannelData(0);
    for (var i=0; i<bufSize; i++) {
      var white = Math.random()*2-1;
      b0=0.99886*b0+white*0.0555179;
      b1=0.99332*b1+white*0.0750759;
      b2=0.96900*b2+white*0.1538520;
      b3=0.86650*b3+white*0.3104856;
      b4=0.55000*b4+white*0.5329522;
      b5=-0.7616*b5-white*0.0168980;
      out[i]=(b0+b1+b2+b3+b4+b5+white*0.5362)*0.11;
    }
  };
  return node;
}
```

### 7.2 呼吸包络

```javascript
function getBreathGain(elapsed) {
  var t = (elapsed % 4000) / 4000; // 0-1, 4秒周期
  if (t < 0.3) return t / 0.3;           // 吸：0→1 (1.2s)
  if (t < 0.4) return 1;                 // 停：1 (0.4s)
  if (t < 0.9) return 1 - (t-0.4)/0.5;  // 呼：1→0 (2.0s)
  return 0;                              // 停：0 (0.4s)
}
```

### 7.3 交叉淡入淡出

阶段切换时，旧层和新层在1.5秒内交叉过渡：

```javascript
function crossfade(oldGain, newGain, duration, ctx) {
  var now = ctx.currentTime;
  oldGain.gain.setTargetAtTime(0, now, duration/3);
  newGain.gain.setTargetAtTime(1, now + duration*0.3, duration/3);
}
```

## 九、总结

这个音频方案本身也是五空方法论的一个实例：

| 概念 | 视觉通道 | 听觉通道 |
|:-----|:--------|:--------|
| 三轮体空 | 粒子消散 | 音调滑升渐弱 |
| 大圆镜智 | 背景镜面 | （视觉专属） |
| 觉知域 | 画布呼吸 | **呼吸层（全程）** |
| 为道日损 | 节点渐隐 | 频率退回低八度 |
| 色空不二 | 粒子上旋 | 泛音短暂闪烁 |

> 呼吸层是整个音频系统的灵魂。它不是"背景音乐"，它是觉知域的听觉化身——你听不到它，但它一直在那里。
